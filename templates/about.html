About

What is it?

Itâ€™s a simple face recognition application with a twist. It finds face in an image uploaded by you and finds a best match from list of celebrities that the application knows about.

The percentage indicates how much the face matches with the famous person. Higher the percentage more similar the faces.

Currently, there are about 150 male and 50 female famous Indian personalities.

How is it done?

There is machine learning at the backend - face recognition to be specific. It uses Face Recognition python library to get face embeddings from uploaded image, which is basically a vector of 128 numbers.

Then simply calculates euclidean distance between known face embeddings and the uploaded face embeddings. Shortest distance taken as the best match.
